# MyProjects

A collection of personal and academic projects built to learn and apply new technologies.

## 🏦 BankingApp
**Full-stack digital banking application**

A complete digital banking platform simulating real-world banking features — built to put C#/.NET and Angular into practice.

**Features:** Account management · Card requests · Transfers (account & IBAN) · Loan applications · Service payments · Notifications · Admin approval workflow · Mobile-first responsive design

**Tech:** .NET 10 · Angular 19 · PostgreSQL · Entity Framework Core · JWT · Docker · Fly.io

🔗 [Live Demo](https://bankingapp-web.fly.dev) · 📂 [Project Folder](./BankingApp)
